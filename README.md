# 📊 Quarterly Employee Performance & Productivity Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze employee performance and productivity across different quarters. The dashboard provides HR managers and business leaders with insights into employee ratings, efficiency, project workload, working hours, and department-wise performance.

The report helps identify top-performing employees, compare estimated versus actual working hours, monitor quarterly trends, and evaluate department performance through interactive visualizations.

---

## 🎯 Business Objective

The objective of this dashboard is to:

- Monitor employee performance across quarters.
- Track productivity and efficiency.
- Compare estimated hours with actual hours worked.
- Identify top-performing employees.
- Analyze department-wise performance.
- Support management in making data-driven decisions.

## 📊 Dashboard Features

### KPI Cards

- Total Employees
- Average Rating
- Average Domain Rating
- Efficiency %
- Total Projects
- Total Actual Hours

---

### Visualizations

#### Employee Performance Ranking
Displays employees ranked according to their average performance rating.

#### Estimated vs Actual Hours
Compares estimated project hours with actual hours worked.

#### Quarterly Rating Trends
Shows rating trends across different quarters.

#### Domain Performance
Analyzes employee performance across different technical domains such as:

- Power BI
- SQL
- Python
- Azure
- Databricks

#### Department-wise Performance
Displays departmental metrics including:

- Average Rating
- Average Domain Rating
- Efficiency %
- Total Actual Hours

#### Quarter Filter
Interactive slicer allowing users to analyze dashboard metrics by quarter.
